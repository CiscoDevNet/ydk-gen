# YDK core and bundle stubs

The `sdk` directory contains the `Python`, `C++` and `Go` code for YDK `core` package

```
sdk
├── _docsgen_common # Common documents for C++, Python, Go
├── cpp
│   ├── core # C++ core
│   |      ├── docsgen # Core documentation
│   |      ├-- src     # Source code
│   |      ├-- samples # Samples
│   |      └── tests   # Core tests
│   ├── install
│   ├── packages
│   ├── samples
│   └── tests # C++ bundle tests
├── go
│   ├── core # Go core
│   |      ├── docsgen # Core documentation
│   |      ├-- samples # Samples
│   |      ├-- tests   # Core & bundle tests
│   |      └── ydk     # Source code
│   └── packages
└── python
    ├── core # Python core
│   |      ├── docsgen # Core documentation
│   |      ├-- samples # Samples
│   |      ├-- tests   # Core & bundle tests
│   |      └── ydk     # Source code
    └── packages

```
