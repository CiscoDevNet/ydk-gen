DefaultRoutingInstance¶

class ydk.models.ietf.ietf_routing.DefaultRoutingInstance¶

Bases: Entity, Identity, RoutingInstance

This class represents configuration data.

This identity represents either a default routing instance, or

the only routing instance on systems that do not support

multiple instances.
